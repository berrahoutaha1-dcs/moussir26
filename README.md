# Commercial Management System

نظام إدارة تجارية سطح المكتب مبني باستخدام Electron + React + SQLite

## المميزات

- واجهة مستخدم حديثة وسهلة الاستخدام
- تصميم متجاوب يعمل على جميع أحجام الشاشات
- دعم اللغة العربية (RTL)
- هيكلة مشروع منظمة تتبع أفضل الممارسات

## التقنيات المستخدمة

- **Electron**: لتطوير تطبيق سطح المكتب
- **React**: مكتبة JavaScript لبناء واجهة المستخدم
- **React Router**: للتنقل بين الصفحات
- **Lucide React**: للأيقونات
- **SQLite**: قاعدة البيانات (جاهزة للربط)

## التثبيت

1. تثبيت المتطلبات:
```bash
npm install
```

## التشغيل

### وضع التطوير
```bash
npm run electron-dev
```

### بناء التطبيق للإنتاج
```bash
npm run build
npm run electron-pack
```

## الصفحات المتاحة

- **لوحة التحكم**: نظرة عامة على النظام والإحصائيات
- **المنتجات**: إدارة المنتجات
- **العملاء**: إدارة قاعدة بيانات العملاء
- **الموردون**: إدارة الموردين
- **المبيعات**: إدارة فواتير المبيعات
- **المشتريات**: إدارة فواتير المشتريات
- **المخزون**: مراقبة وإدارة المخزون
- **التقارير**: عرض وتحليل البيانات
- **الإعدادات**: إعدادات النظام

## ملاحظات

- هذا المشروع يحتوي فقط على الواجهة (UI/Layout) بدون منطق أعمال أو ربط حقيقي مع قاعدة البيانات
- يمكنك إضافة منطق الأعمال وربط قاعدة البيانات SQLite لاحقاً
